# MZQC python library

A python library to use mzQC files. Specifically, have a usable object representation of mzQC that can
* serialise
* deserialise
* syntactic checks
* semantic checks

## Install
```
pip install -e git+'https://github.com/bigbio/mzQC'
```

## MZQC
Schema see https://github.com/HUPO-PSI/mzQC/
